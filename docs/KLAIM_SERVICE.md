# Klaim 플랫폼 서비스 문서

## 개요

**Klaim**은 AI 스타트업을 위한 토큰 기반 결제 플랫폼입니다. AI 애플리케이션의 사용량 기반 결제 모델을 구현할 수 있게 해주는 서비스입니다.

[출처: getklaim.com]

## 핵심 기능

### 1. 토큰 기반 결제 시스템
- AI 앱의 사용량에 따른 크레딧 기반 과금
- 복잡한 결제 인프라 처리 자동화
- Stripe 결제 처리 통합

### 2. 프로모션 페이지 생성
- **Stand-alone Page**: 독립적인 결제 페이지 생성
- **Embedded Widget**: 웹사이트에 임베딩 가능한 위젯

### 3. 개발자 친화적 구현
- "10분 만에" 토큰 기반 결제 시스템 구축
- "0 lines of code" 구현 가능
- API 및 웹훅 통합 지원
